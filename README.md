# React Todo App

A Todo Application built with React, Vite and Tailwind CSS.

## Features

- Add Tasks
- Edit Tasks
- Delete Tasks
- Mark Tasks as Complete
- Show Finished Filter
- Dark Mode
- Local Storage Persistence
- Installable PWA (Progressive Web App)
- Automatic Updates
- Responsive Design


## Tech Stack

- React
- Vite
- CSS
- Tailwind CSS
- JavaScript
- Vite PWA Plugin
